# YoutubeShorts

YoutubeShorts is an application that generates vertical videos from text and background media files.

It produces:
- A **9:16 vertical video**
- **AI-generated voice-over** from input text
- **Subtitles** synchronized with audio
- Combined background visuals from provided media


## How it works
1. Provide a script (text input)
2. Add background image or media files
3. The system processes and composes the video
4. An exported MP4 video is generated


## Output
The output is a fully rendered vertical video ready for platforms such as **YouTube Shorts**, **TikTok**, and **Instagram Reels**.
